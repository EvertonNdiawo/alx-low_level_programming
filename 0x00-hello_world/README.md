This project is aimed at setting up the environment for development in c and getting started with the basics of the language. It contains a variety of tasks for ALX's hello_world project.

0-preprocessor -> a script that runs a C file through the preprocessor and save the result into another file, c. 

1-compiler -> script to a c file but not link. C file is saved in the variable $CFILE and the output should also be named same as the C file, but with the extension .o instead of .c.

2-assembler -> Script that generates the assembly code for a C code and saves it in an output file. The C file name is saved in the variable $CFILE. Output is named same as C file but with the extention .s instead of .c.
