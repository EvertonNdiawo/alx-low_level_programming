This project explores variables, if else statements and while loops in the c programing language.
Below is a detailed outline of what each program does. 

0-positive_or_negative.c -> A program that assigns a random number to a variable n each time it is executed and prints the generated number followed by a statement whether it is positive, negative or is zero.

1-last_digit.c -> A program that assigns a random number to a variable n each time it is executed and prints the last digit of the number stored in the variable.

2-print_alphabet.c -> A c program to print the alphabet in lowercase to the standart output making use of the putchar method as opposed to the printf method. 

3-print_alphabets.c -> A program to print alphabets in lowercase followed by alphabets in uppercase and finally a newline character using the putchar method.

4-print_alphabt.c -> a program that prints the alphabet in lowercase, followed by a newline and prints all the letters except q and eusing the putchar function.

5-print_numbers.c -> A c program that prints all single digit numbers of base 10 starting from 0, followed by a new line.
